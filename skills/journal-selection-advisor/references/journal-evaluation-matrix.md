# Journal Evaluation Matrix

Use this reference to structure journal recommendations after candidate search.

## Scoring Dimensions

Score each candidate from 1 to 5 where useful. Do not pretend the scores are mathematically precise; they are a decision aid.

| Dimension | What to inspect |
|---|---|
| Scope fit | Aims and scope, recent article examples, field audience. |
| Article-type fit | Whether the journal accepts the user's manuscript type. |
| Evidence fit | Whether the manuscript's methods, validation, and novelty match the journal level. |
| Tier fit | JCR/CAS/indexing target, institutional rules, funder requirements. |
| Speed fit | Official timelines, publication model, user's deadline. |
| Cost fit | APC, OA requirement, waiver options, institutional budget. |
| Risk | Warning lists, questionable special issues, unclear indexing, predatory signals. |
| Strategic value | Visibility, audience, career or institutional value. |

## Recommendation Tiers

| Tier | Meaning | Use when |
|---|---|---|
| Reach | Worth trying if the user can tolerate rejection risk. | Manuscript has a credible path but needs polish or stronger framing. |
| Target | Best overall fit. | Scope, article type, evidence, and user goals align. |
| Safe | Lower rejection risk. | Timeline or institutional requirement matters more than prestige. |
| Fallback | Backup route. | Useful when the first 2-3 choices fail or deadline pressure rises. |
| Avoid/Not fit | Do not prioritize. | Scope mismatch, rules mismatch, suspicious signal, or user goal conflict. |

## Output Table

Use a table like this:

| Tier | Journal | Why it fits | Key requirements to verify | Risks | Action |
|---|---|---|---|---|---|
| Target | Journal name | Scope and article type align with the manuscript. | JCR/CAS, APC, article type, word limit. | Moderate rejection risk. | Prepare formatted submission package. |

## Feasibility Language

Use calibrated wording:

- "Realistic if the manuscript strengthens X."
- "A good target based on current evidence."
- "A safer option if the deadline dominates."
- "Not recommended because the scope and article type do not match."
- "Needs verification because current indexing or partition data are not available from the sources checked."

Do not say a journal is guaranteed, easy, or certain.
