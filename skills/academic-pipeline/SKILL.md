# Academic Pipeline Router

Use this skill as an alias for the full ScholarFlow research-to-paper pipeline.

## Route

1. Start with `paper-spine` for research question, contribution, evidence chain, and manuscript architecture.
2. Use `nature-academic-search`, `nature-literature-pipeline`, and `nature-reader` for source discovery and reading.
3. Use `nature-writing`, `nature-polishing`, and `nature-citation` for manuscript drafting, revision, and references.
4. Use `nature-data`, `nature-figure`, and `sci-figure-composer` when results, data checks, or figures drive the paper.
5. Use `nature-reviewer`, `paper-self-review`, `research-integrity-guardrail`, and `nature-response` for review simulation, quality gates, and reviewer response.
