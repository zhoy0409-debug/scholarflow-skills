# Omics Analysis Router

Use this skill for genomics, transcriptomics, metagenomics, microbiome, variant, phylogenomics, and multi-omics analysis requests.

## Route

1. Identify the omics type and data stage: raw reads, alignment, variants, abundance matrix, annotation, statistics, or figures.
2. Load the most specific installed `bio-*`, `biopython-*`, `scikit-bio`, `gget-genomic-databases`, `kegg-*`, or `cobrapy-metabolic-modeling` skill.
3. Use `nature-data` for data audit, reproducibility, statistics, and reporting checks.
4. Use `nature-figure` for publication-grade figures.
5. Use `nature-writing` or `nature-polishing` only after the analysis design/results are clear.
