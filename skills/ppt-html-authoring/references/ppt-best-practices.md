# PPT 制作经验与排版指南

本文档汇总 PPT 制作的专业经验，供 AI 创作 PPT 时参考，确保生成的幻灯片具有专业品质。

---

## 一、字号规范（最重要）

### PPT 字号标准

PPT 字号单位是 **磅数（pt）**。当前画布固定为 1280×720 css px（PowerPoint 现代默认 16:9，等价于 960pt × 540pt），HTML 中直接用 CSS 的 `pt` 单位即可与 PPT 字号一一对应。

| 层级 | PPT 字号 | HTML CSS 字号 | 适用场景 |
|------|----------|---------------|----------|
| 页面主标题 | 24–28pt | 24–28pt | 每页顶部标题，直接表达核心观点 |
| 卡片标题（一级） | 14–18pt | 14–18pt | 卡片区域标题、分区标题 |
| 内容正文 | 12–14pt | 12–14pt | 主要内容文本 |
| 二级内容 | 11–12pt | 11–12pt | 子内容、补充说明 |
| 标签/注释 | 9–10pt | 9–10pt | 说明性文字、脚注 |
| 最小字号 | 8pt | 8pt | 极少数注释，不建议更小 |

> **关键原则**：宁可字大内容少，也不要字小内容多。信息过载比留白更糟糕。

### 字号黄金比例

- **标题与正文比例**：建议 3:2（如标题 24pt，正文 14pt）
- **层级递减**：每降一级，字号减少 2–4pt
- **单页字号种类**：不超过 3 种

### 常见字号错误

| 错误 | 正确做法 |
|------|----------|
| 卡片标题用 11pt（太小） | 卡片标题至少 14pt |
| 正文用 9–10pt（太小） | 正文至少 12pt |
| 结尾金句用 11pt（太小） | 金句至少 36pt，甚至 48pt |
| 所有文字大小一样 | 建立清晰的字号层级 |
| 页面标题用 14pt（不够醒目） | 页面标题至少 20pt |

### 特殊页面字号建议

| 页面类型 | 主标题 | 副标题/正文 | 说明 |
|----------|--------|-------------|------|
| 封面页 | 36–48pt | 18–24pt | 标题要大气醒目 |
| 目录页 | 24–28pt | 14–16pt | 章节标题清晰可辨 |
| 内容页 | 20–24pt | 12–14pt | 标题观点鲜明 |
| 数据页 | 20–24pt | 36–48pt（数字） | 关键数字要大 |
| 金句/结尾页 | 36–48pt | 14–18pt | 金句是视觉焦点 |
| 架构图 | 20–24pt | 10–12pt | 模块文字可适当小 |

## 二、排版原则

### 核心法则

1. **减法思维**：将内容精简至原计划的 60%，每页聚焦一个核心观点
2. **留白即设计**：留白不是浪费空间，而是引导视线的设计元素
3. **对齐是基础**：所有元素必须严格对齐，使用网格系统
4. **一致性**：全文统一字体、配色、间距、卡片样式

### 布局参考

- **内容区域**：距离幻灯片边缘保持 30–40pt 的边距
- **标题区域**：页面顶部，高度约 50–60pt
- **内容区域**：标题下方到页面底部，留出 20–30pt 底部边距
- **分栏间距**：栏与栏之间保持 10–15pt 间距

### 信息密度控制

| 原则 | 说明 |
|------|------|
| 每页 ≤ 3 个要点 | 超过 3 个则拆分为多页 |
| 文字行数 ≤ 7 行 | 大段文字应精简为要点 |
| 图文比例 | 文字占 40–60%，图/图表/留白占 40–60% |
| 避免满屏文字 | 用图标、色块、卡片分区组织信息 |

## 三、色彩使用经验

### 配色原则

- **三色原则**：主色 + 辅色 + 点缀色，不超过 3 种主色
- **60-30-10 法则**：60% 主色（背景/大面积），30% 辅色（内容区），10% 点缀色（强调）
- **低饱和度**：大面积色块使用低饱和度色彩，减少视觉疲劳
- **高对比度**：文字与背景的对比度至少 4.5:1

### 配色使用要点（通用建议）

具体品牌色由所在项目决定。下表给出**结构性**建议（哪些场景用哪类色彩），具体十六进制值请按你的品牌色板替换。

| 场景 | 推荐色类 | 注意事项 |
|------|----------|----------|
| 标题强调 | 品牌强调色（红/橙/紫等暖色） | 不要大面积使用强调色，易造成视觉混乱 |
| 正文文字 | 黑色 #000000 或深灰 #333333 | 纯黑在白底上刺眼，深灰更舒适 |
| 辅助文字 | 灰色 #8C8C8C | 注释、非重点内容 |
| 卡片背景 | 浅灰 #F2F2F2 | 区分内容区域 |
| 图表配色 | 品牌主色 / 蓝色系 | 数据展示首选 |
| 正向指标 | 绿色 #61B230 | 增长、达标 |
| 警示信息 | 红色 / 强调色 | 风险、未达标 |

## 四、视觉元素经验

### 卡片设计

- 卡片是企业风 PPT 的核心组件，用于信息分区
- 卡片内边距：上下 8–12pt，左右 10–15pt
- 卡片间距：8–12pt
- 卡片圆角：建议 4–6pt
- 卡片边框：0.5pt 宽度

### 图标使用

- 图标大小：建议 30–40pt（正文旁），20–24pt（标签旁）
- 图标颜色：与对应文字或主题色一致
- 图标间距：与文字保持 5–8pt 间距

### 数字展示

- 关键数字要大（36–48pt），配合单位（14–18pt）
- 数字颜色使用品牌强调色或蓝色
- 数字下方配简短说明文字（10–12pt）

## 五、页面类型制作要点

### 封面页

- 标题居中或偏左，字号 36–48pt
- 副标题 18–24pt
- 底部可放日期、部门信息（12–14pt）
- 背景简洁，可用纯色或极简图形

### 目录页

- 章节编号醒目（24–36pt）
- 章节标题 16–20pt
- 当前章节可用红色高亮
- 布局清晰，留白充足

### 内容页（图文/分栏）

- 页面标题 20–24pt，直接表达观点
- 卡片标题 14–18pt
- 正文 12–14pt
- 每个卡片聚焦一个要点

### 数据展示页

- 关键数字要大（36–48pt）
- 配合同比/环比箭头
- 图表标题 14–16pt
- 数据标签 10–12pt

### 流程图/架构图

- 步骤标题 12–14pt
- 步骤内容 10–12pt
- 连接线清晰，箭头方向明确
- 层次分明，颜色区分不同层级

### 金句/结尾页

- 金句文字 36–48pt，居中
- 可使用品牌强调色或深色背景反白
- 配合简短署名或出处（14–16pt）
- 整体留白充足，突出金句

## 六、自检清单

AI 生成 PPT 后，逐项检查：

### 字号检查
- [ ] 页面标题是否 ≥ 20pt？
- [ ] 卡片标题是否 ≥ 14pt？
- [ ] 正文是否 ≥ 12pt？
- [ ] 金句/结尾文字是否 ≥ 36pt？
- [ ] 关键数字是否 ≥ 36pt？
- [ ] 最小字号是否 ≥ 8pt？

### 排版检查
- [ ] 每页要点是否 ≤ 3 个？
- [ ] 元素是否对齐？
- [ ] 留白是否充足？
- [ ] 字号种类是否 ≤ 3 种？

### 配色检查
- [ ] 主色是否统一？
- [ ] 文字与背景对比度是否足够？
- [ ] 强调色使用是否克制？

### 内容检查
- [ ] 标题是否直接表达观点？
- [ ] 是否有多余的装饰元素？
- [ ] 图标是否合理且必要？
