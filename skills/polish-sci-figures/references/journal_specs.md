# Journal-specific figure requirements

This skill does not treat a static table as an authority. Journal figure rules
change and may differ by article type. For a submission, retrieve the current
official author instructions and the target manuscript template before setting
width, font, resolution, file format, color mode, or panel-label convention.

Record the official source URL and retrieval date with the submission files.
Where the journal is unknown, retain a vector master and high-resolution PNG,
use readable typography, and defer final dimensions until a target is chosen.

## What to verify from the official source

- single- and double-column width, maximum height, and whether a composite is allowed
- minimum figure-text size at final print width and required font family
- raster resolution by image type, colour profile, and TIFF/EPS/PDF/SVG policy
- panel-label convention, figure-legend placement, and supplementary-figure rules
- source-data, image-integrity, and accessibility requirements

For a public showcase, use the `showcase` rules in `SKILL.md`, not journal submission rules.
