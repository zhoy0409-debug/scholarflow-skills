# 异常记录

记录实验过程中发现的所有异常，供后续质量控制和故障排查。

## 格式

```yaml
---
date: YYYY-MM-DD
related_exp: EXP-ID
anomaly_type: 设备故障 / 操作失误 / 结果异常 / 安全事件
severity: 低 / 中 / 高
status: 待处理 / 已处理 / 需复查
---

## 描述

[异常现象的客观描述]

## 影响

[对实验结果的影响评估]

## 处理

[已采取的措施]

## 备注

[后续建议]
```

## 使用

- 每次实验结束后检查是否有异常
- 有则追加记录到此文件
- 定期复查未处理异常
