# Scenario: Conference

Research conferences as deadline-bound peer-review venues.

## Official Sources

Find and record:

- call for papers,
- submission guidelines,
- paper template,
- page limits,
- anonymity rules,
- supplementary material rules,
- artifact/code policy,
- review criteria,
- accepted paper examples or proceedings.

## Example Learning

Learn:

- how accepted papers frame novelty quickly,
- how related work is compressed,
- how experiments are selected under page limits,
- how contributions are stated,
- how limitations are handled without weakening the core claim.
