# AI Check Router

Use this skill for AI-writing risk checks, disclosure review, citation integrity, and manuscript-quality audits.

## Route

1. Use `paper-spine-humanize` when the user asks about AI-like phrasing or humanization diagnostics.
2. Use `paper-self-review` for manuscript self-review.
3. Use `nature-citation` for citation integrity and reference verification.
4. Use `research-integrity-guardrail` for disclosure, integrity gates, unsupported claims, and academic-paper workflow checks.

Prefer transparent revision notes and preserve the user's actual scientific meaning.
